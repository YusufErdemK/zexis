<h1 align="center">ZeXis OS</h1>
An open-source operating system based on <strong>Zorin OS</strong> designed for efficiency and customization.

<img src="readme-files/image1.png" style="margin-top: 30px;">

## Features
- **User-Friendly Interface**: Intuitive design for both beginners and advanced users.
- **Lightweight**: Optimized for performance on older hardware.
- **Customizable**: Extensive options for personalizing the desktop environment.
- **Pre-installed Applications**: Comes with essential software for productivity and entertainment.
- **Regular Updates**: Frequent updates to ensure security and add new features.

## Installation

To install ZeXis OS, follow these steps:

1. Download the latest ISO's torrent from the [Releases](https://github.com/YusufErdemK/zexis/releases) page.

2. Install the ISO with qBitTorrent or any torrent client of your choice.

3. Create a bootable USB drive using tools like Rufus or Etcher.

4. Boot your computer or VM from the USB drive and follow the on-screen instructions to install ZeXis OS.
